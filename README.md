# Auto PC Cleaner (Batch Script) 🚀

Skrip sederhana berbasis Windows Batch (`.bat`) untuk membersihkan file sampah (*junk files*), mengoptimalkan ruang penyimpanan, dan menyegarkan koneksi jaringan secara otomatis.

## 📋 Deskripsi
Skrip ini dirancang untuk melakukan pembersihan rutin pada sistem Windows tanpa harus membuka berbagai menu secara manual. Sangat berguna untuk menjaga performa laptop/PC agar tetap ringan, terutama bagi pengguna yang sering berurusan dengan pengolahan data atau pengembangan aplikasi.

## 🛠️ Fitur Pembersihan
Skrip ini mencakup 6 tahap pembersihan utama:
1.  **User Temporary Files:** Menghapus file sampah dari aplikasi pihak ketiga.
2.  **Windows System Temp:** Membersihkan file sementara milik sistem operasi.
3.  **Prefetch Cache:** Mengoptimalkan catatan peluncuran aplikasi.
4.  **Windows Update Cache:** Menghapus sisa unduhan update Windows yang berukuran besar.
5.  **DNS Flush:** Menyegarkan cache DNS untuk koneksi internet yang lebih stabil.
6.  **Recycle Bin:** Mengosongkan tempat sampah secara permanen.

## 🚀 Cara Penggunaan
1.  Unduh file `cleaner.bat` (atau nama file yang Anda buat).
2.  Klik kanan pada file tersebut.
3.  Pilih **Run as Administrator** (Wajib, agar skrip memiliki izin menghapus file sistem).
4.  Tunggu hingga proses selesai dan tekan tombol apa saja untuk keluar.

## ⚠️ Catatan Penting
- Pastikan semua pekerjaan atau dokumen sudah disimpan sebelum menjalankan skrip.
- Beberapa file mungkin tidak terhapus jika aplikasi terkait sedang terbuka (skrip akan otomatis melewati file tersebut).
- Disarankan untuk menjalankan skrip ini seminggu sekali.
